# Welcom to API document powered by ReDoc

## How to Start

### Start Server

```shell
git clone https://github.com/hokan-inc/hokan_api_document.git
cd hokan_api_document
yarn install
yarn run develop
```

and open http://localhost:9090

## How to build

```shell
yarn run build
```
`public` is output directory
